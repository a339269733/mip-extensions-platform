## 示例

### 基本用法

```html
<mip-hunliji-integralstatistic
  data-api="https://mtest.hunliji.com/p/wedding/index.php/Home/APIUserGrow/shareScore"

```

## 属性

### {data-api}

说明：{分享接口地址}
必选项：{是}
类型：{string}

### {data-code}

说明：{分享操作类型}
必选项：{是}
类型：{string}

### {data-model}

说明：{分享内容类型}
必选项：{是}
类型：{string}
